🚑 快速修复工具

正在检测问题...

🔧 自动修复步骤

  1. 清除Service Worker - 移除可能损坏的缓存服务
  2. 清除所有缓存 - 删除浏览器缓存的旧文件
  3. 重置本地存储 - 清除可能损坏的应用数据
  4. 强制刷新应用 - 重新加载最新版本

🛠️ 手动修复步骤

  1. 打开浏览器开发者工具
    • 按 F12 或右键选择"检查"
  2. 清除Service Worker
    • 转到 Application 标签页
    • 在左侧找到 Service Workers
    • 点击 "Unregister" 注销所有Service Worker
  3. 清除存储数据
    • 在 Application 标签页中找到 Storage
    • 点击 "Clear storage"
    • 确保选中所有选项，然后点击 "Clear site data"
  4. 硬刷新页面
    • 按 Ctrl+Shift+R (Windows) 或 Cmd+Shift+R (Mac)
    • 或者按住 Shift 键点击刷新按钮
